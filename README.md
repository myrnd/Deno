# Deno
Typescript and Javascript Secure Runtime

## Deno Shell Commands
* `deno` to start Deno shell
* `console.clear()` to clear shell
* `ctrl+d` to close Deno Shell
* `deno help` to get help on Deno commands

## Deno Global Objects
* `Deno`
** `Deno.exit()`
** `Deno.version`
* `window`
** `window.console`
